## 一、一句话定义（Elevator Pitch）

> **「距离即是命运，走位决定生死。一款以三轨道位置战斗为核心、迷雾棋盘探索为驱动、恶魔交易改写规则的Roguelite卡牌游戏。」**

---

## 二、核心体验目标

玩家在每一局结束时应该说出：
- **「我怎么把游戏玩成这样的？」**（意外性）
- **「下一局我要试试这个组合」**（重复可玩性）
- **「这个角色和上个角色完全不一样」**（角色差异化）

---

## 三、核心循环（The Core Loop）

```
[选择角色]
    ↓
[迷雾棋盘探索] — 掷骰/消耗探索资源，决定前进步数与方向
  ├─ 迷雾下可能是：战斗 / 精英 / 商店 / 事件 / 宝箱 / Boss
  └─ 核心张力：求稳打Boss  vs  贪婪探迷雾
    ↓
[三轨道回合制卡牌战斗]
  ├─ 距离变效牌 + 基础牌
  ├─ 免费移动权保证操作空间
  └─ 击败敌人 → 选择奖励
        ├─ 新卡牌（扩充手段）
        └─ 进入商店机会
    ↓
[双轨商店]
  ├─ 常规商店（金币）→ 基础卡牌 / 删牌 / 数值强化
  └─ 恶魔商店（消耗最大生命值上限）→ 颠覆性Rule Mutators
    ↓
[道具叠加，规则越来越扭曲]
    ↓
[Boss战] → 进入下一层
    ↓
[死亡 or 通关] → 解锁新变效牌/新恶魔道具池 → 开始新局
```

## 四、战斗系统（三轨道·骰子·距离变效）

> **核心张力：距离即是命运，走位决定生死。**

### 战场结构
```
[远程轨道] ←————— [中程轨道] ←————— [近程轨道]
    你                                   敌人
```
玩家与敌人的距离是战局最核心的变量。

---

### 回合流程
```
① 敌方明牌（系统掷骰）
   每回合开始，系统掷骰决定敌人意图（提前公示）
   骰面 1-2 → 逼近（向玩家靠近一条轨道）
   骰面 3-4 → 原地蓄力（下回合伤害翻倍）
   骰面 5-6 → 高危攻击（当前轨道大伤害）

② 战术规避（移动掌控）
   玩家拥有基础移动权：每回合1次免费移动
   移动不占用手牌资源 → 始终有操作空间和规避希望

③ 精准打击（出牌结算）
   根据当前轨道打出手牌
   UI 高亮当前轨道可触发的效果（所见即所得）

④ 回合结束
   敌人执行意图，格挡清零，进入下一回合
```

---

### 卡牌比例：主次分明，降低认知过载

| 类型 | 占比 | 定义 |
|------|------|------|
| **基础牌** | 70% | 任意轨道效果稳定，提供数值下限 |
| **距离变效牌** | 30% | 不同轨道效果完全不同，提供策略上限 |

---

### 距离变效牌示例

| 卡牌 | 费用 | 远程效果 | 中程效果 | 近程效果 |
|------|------|----------|----------|----------|
| **黑影** | 1 | 隐身，下次攻击+50%伤害 | 闪现，瞬移至任意轨道 | 背刺，12伤 |
| **烈焰** | 1 | 发射火球，8伤+灼烧 | 火浪横扫，6伤全体 | 近身爆炸，5伤+击退敌人 |
| **护盾祈祷** | 1 | 获得10格挡 | 获得6格挡+抽1张 | 获得4格挡+反伤3 |

---

### 状态系统
| 状态 | 效果 |
|------|------|
| 灼烧 | 每回合结束受伤 |
| 虚弱 | 攻击伤害降低25% |
| 易伤 | 承受伤害增加25% |
| 中毒 | 每回合受伤，层数递减 |
| 定身 | 本回合无法移动 |
| 蓄力 | 下回合第一张牌效果翻倍 |

## 五、道具系统（Rule Mutators）——核心差异点

> **道具不改变数值，改变规则本身。**

### 道具设计原则
每个道具必须回答：**「它让哪张牌变得奇怪？」**

### 示例道具（约60个为MVP目标）

| 道具名 | 规则变异效果 |
|--------|------------|
| **镜像碎片** | 每次打出攻击牌，同时触发一次你最后打出的防御牌 |
| **血契** | 每次受到伤害，手中所有牌费用-1（本回合） |
| **空洞之眼** | 打出的牌不进入弃牌堆，直接消除（Exhaust） |
| **回响石** | 每回合第一张牌打出两次 |
| **寄生虫** | 每次抽牌时，额外抽1张，但牌组中加入1张「腐化」 |
| **倒计时** | 手牌中每张牌每回合费用-1，到0时自动触发 |
| **捆缚之链** | 防御牌同时对攻击该目标造成等量伤害 |
| **混沌核心** | 每回合开始，随机一张手牌变成随机卡牌 |
| **幸存者本能** | 生命低于30%时，每回合开始自动打出手中费用最低的牌 |
| **记忆碎片** | 使用过的卡牌效果，下回合有20%概率再次触发 |

### 道具稀有度
- **普通**：简单规则扭曲
- **罕见**：中等复杂度变异
- **诅咒道具**：强力但附带永久负面规则

## 六、角色设计框架

> **每个角色 = 专属起始牌组 + 一个独特核心机制柱**

### MVP阶段目标：3个角色

---

### 角色A：「遗弃者」(The Forsaken)
**视觉参考**：赤裸的孩子，以眼泪为武器（致敬以撒）
**核心机制柱**：**「积累」**——卡牌打出次数越多，特定效果叠加

```
专属机制：「凝聚」(Convergence)
  每打出一张攻击牌，获得1层「凝聚」
  凝聚可以被某些牌消耗，产生爆发效果
```

**起始牌组（10张）**：
- 泪弹 ×4（1费，造成6伤害）
- 护盾 ×4（1费，获得5格挡）
- 凝聚爆发 ×1（2费，消耗所有凝聚，每层造成3伤害）
- 内省 ×1（0费，抽2张牌）

---

### 角色B：「堕落天使」(The Fallen)
**视觉参考**：断翼的黑暗天使
**核心机制柱**：**「代价」**——强力效果必须用生命值或格挡来购买

```
专属机制：「献祭」(Sacrifice)
  部分卡牌标注「献祭X」：消耗X点生命/格挡来强化效果
  拥有独特资源「黑焰」，受伤时积累，可增幅献祭效果
```

**起始牌组（10张）**：
- 堕落之刃 ×3（1费，造成8伤害；献祭3生命→造成16伤害）
- 黑暗护甲 ×3（1费，获得7格挡；献祭4格挡→再获得12格挡）
- 血之代价 ×2（0费，失去4生命，抽3张牌）
- 黑焰积累 ×2（0费，积累2层黑焰）

---

### 角色C：「机械先知」(The Oracle)
**视觉参考**：古代机械生命体，参考Defect
**核心机制柱**：**「预言」**——可以预先设定下回合的效果，形成连锁

```
专属机制：「预言槽」(Prophecy Slot)
  拥有2个预言槽，可存放「延迟效果」
  下回合开始时自动触发所有预言槽中的效果
```

**起始牌组（10张）**：
- 预言打击 ×3（1费，造成5伤害，将「+3伤害」存入预言槽）
- 预言护盾 ×3（1费，获得4格挡，将「+4格挡」存入预言槽）
- 触发预言 ×2（1费，立即触发所有预言槽并清空）
- 数据分析 ×2（0费，查看牌组顶3张，重新排列顺序）

## 七、探索地图：迷雾棋盘（Fog Grid）

> **核心驱动力：贪婪与恐惧的反复横跳。**

### 地图结构
```
迷雾覆盖的网格区域（例：8×8格）
玩家起点在左下角，Boss区域在右上角

[ ? ][ ? ][ ? ][ ? ][ B ]
[ ? ][ ? ][ ? ][ ? ][ ? ]
[ ? ][ ? ][ ★ ][ ? ][ ? ]  ← 当前位置
[ ? ][ ? ][ ? ][ ? ][ ? ]
[起点][ ? ][ ? ][ ? ][ ? ]

揭开迷雾后显示节点类型
```

### 探索机制
- 每次探索：掷骰（或消耗「探索资源」）决定**前进步数和可选方向**
- 玩家在可达格子中**主动选择**落点
- 迷雾下方的内容直到踩上才揭示

### 节点类型
| 节点 | 描述 |
|------|------|
| ⚔️ 战斗 | 普通敌人，获得卡牌奖励 |
| 💀 精英 | 强化敌人，不可逃脱，必掉Rule Mutator |
| 🏪 常规商店 | 消耗金币购买卡牌/删牌/数值强化 |
| 😈 恶魔商店 | 消耗**最大生命值上限**，购买颠覆性Rule Mutators |
| ❓ 随机事件 | 文字叙事，做出选择 |
| 📦 宝箱 | 直接获得道具 |
| 👑 Boss | 每层终Boss，掉落稀有Rule Mutator |

### 核心博弈
```
安全路线：直奔Boss，保存血量，但错过强力道具
贪婪路线：探更多迷雾，可能触发精英或强制战斗
          但也可能拿到改变整局走向的恶魔道具

玩家每一步都在做「值不值得冒险」的决策
```

## 八、双轨商店系统

### 常规商店（金币驱动）
- 购买基础卡牌
- 删除卡牌（精简牌组）
- 购买数值强化（生命上限+、能量+等）
- 购买消耗品（单次使用的战斗道具）

### 恶魔商店（生命上限驱动）⚠️
> **「你用命换的东西，才是真正能改变命运的东西。」**

- **只售颠覆性Rule Mutators**，不卖基础数值
- 代价：永久降低最大生命值上限（例：-10上限换取一个道具）
- 出现条件：击败精英或Boss后概率触发
- Rule Mutator示例：

| 道具 | 规则改写效果 |
|------|------------|
| **混沌骰** | 每回合开始强制重新掷一次敌人意图骰 |
| **镜像碎片** | 每次打出攻击牌，同时触发上一张防御牌效果 |
| **命运之链** | 你与敌人共享格挡：你获得格挡时敌人也获得格挡，但你受伤时敌人也受伤 |
| **倒计时核心** | 手牌中所有牌每回合费用-1，归零时自动触发 |
| **空洞之眼** | 打出的牌直接消除，不进弃牌堆（Exhaust） |

## 九、开发红线（绝对不做）

> 作为独立开发者，以下内容**敢碰就砍**：

| 红线 | 替代方案 |
|------|----------|
| ❌ 复杂剧情与NPC对话演出 | 叙事全部交给卡牌风味文本（Flavor Text）和恶魔商店的碎片化描述 |
| ❌ 局外永久数值成长（天赋树、属性点） | 重复游玩动力纯靠解锁「新变效牌」和「新恶魔道具池」 |
| ❌ 大地图角色行走动画与碰撞 | 迷雾棋盘保持桌面抽象感，格子即节点，无行走动作 |
| ❌ 5个以上角色（MVP阶段） | 先做3个角色，每个机制清晰，验证后再扩展 |
| ❌ 多人模式 | 无 |

## 九、核心风险与对策

| 风险 | 对策 |
|------|------|
| 道具规则叠加产生无法预期的BUG | 每个道具独立测试，建立「规则冲突优先级」文档 |
| 卡牌数量不足导致每局体验重复 | MVP先做60张，验证核心后扩展 |
| 单人开发美术资产量过大 | 采用简洁像素风或极简几何风，降低美术门槛 |
| 角色机制过于复杂导致开发超时 | 每个角色机制只有「1个关键词」，不做复合机制 |
