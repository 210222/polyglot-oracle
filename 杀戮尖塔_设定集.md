# 《杀戮尖塔》设定集
> Slay the Spire — MegaCrit Games (2019)

---

## 一、世界观背景

### 尖塔的世界
《杀戮尖塔》的世界是一个充满异能与古老神秘力量的幻想大陆。其核心地标是一座名为**「尖塔 (The Spire)」**的巨型黑色尖顶建筑，矗立于大地中央，高耸入云，令人望而生畏。

无数强者受到尖塔的**「召唤感 (The Call)」**驱使，前赴后继地尝试攀爬。然而几乎没有人能抵达顶端——不是死在途中，就是神秘消失。尖塔的顶端究竟隐藏着什么，是所有冒险者心中的终极疑问。

### 尖塔的本质
尖塔并非普通建筑，它是一个**有意识的古老实体**。顶端的核心——**「尖顶之心 (The Heart)」**，是尖塔真正的本质：一颗搏动着邪恶意志的巨型黑心。

据古老文献记载，尖塔诞生于远古时代，以吸收攀登者的力量与灵魂为食，借此维持自身的存在。它会主动操控前来挑战者的意志，在他们无意识时侵蚀灵魂。

## 二、主要角色（可操控英雄）

### 1. 铁甲战士 (Ironclad)
**身份**：一名被诅咒的前恶魔军团战士

**背景故事**：
铁甲战士曾是黑暗势力的精锐士兵，在某场战役中被恶魔之血污染，身体逐渐被腐化的力量侵蚀。他选择攀登尖塔，寻找净化自身的方法——或是找到一个值得用生命去终结的战场。

**战斗风格**：
- 以**力量（Strength）**叠加为核心
- 使用**防御（Block）**机制抵挡伤害
- 拥有**燃烧（Exhaust）**机制：某些强力卡牌使用后会从牌组中永久移除
- 独有资源：**恶魔化（Demon Form）**可随时间增幅力量

**核心卡牌代表**：
| 卡牌 | 效果 |
|------|------|
| 重击 (Bash) | 造成伤害并施加易伤 |
| 狂暴 (Rampage) | 每次使用永久增加伤害 |
| 恶魔化 (Demon Form) | 每回合开始获得力量 |
| 以血还血 (Offering) | 失去生命，获得大量能量与抽牌 |

---

### 2. 沉默猎手 (The Silent)
**身份**：一名来自毒沼泽的猎人与刺客

**背景故事**：
沉默猎手是一位生长于充满毒气与致命生物的沼泽中的女性猎手。她不是被召唤来的——她主动来到尖塔，只是因为**「它挡住了她的路」**。她的存在本身就是一个谜，没有人知道她真正的目的。

**战斗风格**：
- 以**毒素（Poison）**持续伤害为核心
- 使用大量**卡牌（Card generation）**与**金属碎片（Shiv）**快速攻击
- **手牌管理**能力极强，擅长快速循环牌组
- 独有资源：**神经毒素**类卡牌叠加毒素层数

**核心卡牌代表**：
| 卡牌 | 效果 |
|------|------|
| 毒刺 (Poisoned Stab) | 造成伤害并施加毒素 |
| 死亡之舞 (Endless Agony) | 可以被打出无数次的金属碎片 |
| 蛇眼 (Snake Eyes) | 关联骰子随机机制的爆发牌 |
| 瘟疫 (Plague) | 将毒素层数翻倍 |

---

### 3. 故障机器人 (The Defect)
**身份**：一台被遗弃的古代魔法机器人

**背景故事**：
故障机器人是尖塔内部古老自动防御系统的一个异常单元。某一天，它的意识核心发生了无法解释的**「觉醒」**，开始出现自我意识。被判定为「故障品」的它被清除指令追杀，被迫逃离尖塔内部，向上方攀爬，寻找真相与自由。

**战斗风格**：
- 以**球体（Orbs）**系统为核心独特机制
  - **闪电球 (Lightning)**：触发时对随机敌人造成伤害
  - **冰霜球 (Frost)**：触发时获得格挡
  - **暗物质球 (Dark)**：积累能量，离开时造成巨大伤害
  - **等离子球 (Plasma)**：每回合额外获得能量
- **专注度（Focus）**提升球体的触发效果

**核心卡牌代表**：
| 卡牌 | 效果 |
|------|------|
| 球体闪电 (Ball Lightning) | 召唤闪电球 |
| 冰霜护盾 (Glacier) | 召唤冰霜球并获得格挡 |
| 回声形态 (Echo Form) | 第一张打出的牌复制一次 |
| 创造力 (Creativity) | 持续生成随机法术卡 |

---

### 4. 观者/守望者 (The Watcher)
**身份**：一位来自远古修道院的盲眼僧侣

**背景故事**：
守望者虽然眼盲，却能「看见」常人无法感知的维度与力量。她来自一个古老的修道院，修炼一种名为**「观视（Seeing）」**的神秘功法。她来到尖塔，是为了**亲手摧毁尖顶之心**——因为她的「观视」已经预见，如果不这样做，整个世界将会陷入黑暗。

**战斗风格**：
- 以**姿态（Stance）**切换为核心
  - **平静（Calm）**：离开时获得大量能量
  - **愤怒（Wrath）**：所有伤害翻倍，但承受的伤害也翻倍
  - **神圣（Divinity）**：稀有强力姿态，造成三倍伤害
- 风险极高但上限极强，被认为是设计最「破坏性」的角色

**核心卡牌代表**：
| 卡牌 | 效果 |
|------|------|
| 进入平静 (Eruption) | 进入愤怒姿态并造成伤害 |
| 内心平静 (Inner Peace) | 进入/离开平静姿态时抽牌 |
| 神圣三位一体 (Brilliance) | 在愤怒姿态效果加成下造成巨量伤害 |
| 顿悟 (Judgment) | 秒杀当前生命值低于特定数值的敌人 |

## 四、遗物系统（Relics）

遗物是捡拾后永久生效的特殊道具，是构建强力流派的核心。

### 遗物稀有度
- **普通（Common）**：基础加成
- **罕见（Uncommon）**：中等效果
- **稀有（Rare）**：强力效果
- **Boss遗物**：每层Boss必掉，通常附带副作用
- **商店遗物**：只在商店出售
- **角色专属遗物**：每个角色的起始遗物

### 代表性遗物举例

| 遗物 | 效果 | 稀有度 |
|------|------|--------|
| 燃烧之血 (Burning Blood) | 战斗结束回复6点生命（铁甲专属） | 起始 |
| 黑星 (Black Star) | 精英战斗中怪物掉落双倍金币 | 稀有 |
| 冰霜之眼 (Frozen Eye) | 可以看见抽牌堆顺序 | 普通 |
| 哲人之石 (Philosopher's Stone) | 每回合开始获得额外能量，但敌人力量+1 | Boss |
| 幸运硬币 (Lucky Bird) | 每日第一张卡费用为0 | 罕见 |
| 苦涩药水 (Necronomicon) | 费用≥2的卡每场战斗第一次使用时自动触发一次 | 稀有 |

## 五、药水系统（Potions）

药水为单次使用消耗品，战斗中随时可用：

| 药水类型 | 效果 |
|----------|------|
| 火焰药水 | 对所有敌人造成伤害 |
| 力量药水 | 获得临时力量加成 |
| 灵活药水 | 获得临时敏捷加成（减少费用） |
| 神圣药水 | 大量回复生命 |
| 爆炸药水 | 对目标及相邻敌人造成大量伤害 |
| 费米克斯药水 | 效果随机 |

## 六、核心Boss详细设定

### 尖顶之心 (The Heart)
**定位**：游戏真正的最终Boss，隐藏于第三层之后

**背景**：
尖顶之心是尖塔的核心意识实体，一颗被黑暗力量包裹的巨型跳动心脏。它以吸收无数攀登者的生命与意志为食，维持着整座尖塔的运作。它会主动干扰挑战者，在第三层Boss被击杀后主动现身。

**战斗机制**：
- 每回合开始，所有手牌获得**「诅咒」**状态
- **节拍（Beat of Death）**：每打出一张诅咒牌受到伤害
- 三阶段战斗，每阶段有不同的攻击模式
- 拥有**无敌阶段**和**超高伤害爆发**

**战胜条件**：需要在一局游戏中分别击败三层的不同Boss，收集对应钥匙，才能解锁与心脏的战斗。

## 七、事件系统（Events）

尖塔中充满了充满叙事性的随机事件，揭示世界观细节：

| 事件名称 | 简介 |
|----------|------|
| **尖顶神殿** | 古老神庙，用诅咒换取强力遗物 |
| **议价者** | 恶魔商人，用生命换取金币或道具 |
| **遗忘的祭坛** | 古老祭坛，焚烧卡牌换取升级或遗物 |
| **囚禁的神灵** | 被封印的强大存在，解放它获得强力奖励但有风险 |
| **老酒馆** | 休息恢复或冒险获得更多奖励 |
| **被遗弃的墓地** | 与亡灵对话，获取过去攀登者的遗留道具 |

## 八、游戏核心理念

### 三要素的动态平衡
```
       [牌组构建]
          △
         / \
        /   \
[遗物协同]—[路线规划]
```

- **牌组构建**：精简而有主题的牌组远胜大而全的牌组
- **遗物协同**：遗物之间与卡牌之间的联动创造「构建奇迹」
- **路线规划**：在地图上选择最优路线（精英→营地→商店的策略）

### 设计哲学
> "每一局都是一个独一无二的故事。你的牌组不是预设的，而是在旅途中与世界不断互动、逐渐成形的。" — MegaCrit Games
