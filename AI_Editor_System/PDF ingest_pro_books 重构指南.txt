Theoretical Architecture: The Dual-Track ProtocolTo address the limitations of the legacy system while adhering to the user's constraints, we introduce the Dual-Track Content Extraction & Enhancement Architecture. This methodology, derived from the Scanned PDF Data Processing & Storage research, fundamentally shifts the processing paradigm from "Text Reading" to "Visual Decoding".3.1 The Double-Helix Processing Model传统OCR将文档视为线性字符流。双轨架构将文档视为“可视化知识图谱”，将布局结构与语义意义分离。A轨道：结构骨架（PaddleOCR）该轨道使用了PaddleOCR的PP-Structure模块（基于PicoDet）。其主要作用不仅是阅读文本，还包括理解页面的几何结构。它将页面区域分类为离散类型：正文：标准段落要求高精度OCR。标题：用于元数据分割的层级标记。页眉/页脚：必须通过手术去除噪声元素，以防止RAG系统中的上下文污染。图表/表格：复杂的视觉元素，传统OCR会将其扭曲成无法理解的字符流。轨道B：语义灵魂（视觉语言模型）该轨道针对“多模态信息黑洞”。当轨道A识别A或时，将坐标有效载荷交给轨道B。使用视觉语言模型（VLM），如GPT-4o视觉，轨道B生成视觉元素的语义描述。FigureTable对于表格：VLM 将行/列结构重建为 Markdown。图表方面：VLM提取数据趋势（例如，“第三季度收入增长15%”），这些趋势在视觉上存在但文本中缺失。3.2 融合点：“富文本”流Track A（清理后的文本）和Track B（语义描述）的输出在数据融合层中汇聚。在这里，文档被重建为一个顺序的“富文本”块流。该流使下游的 KnowledgeIngester 能够将已扫描的 PDF 完全视为数字文本文件，满足“生成数据库文件不变”的要求。4. 计算机视觉预处理流程从扫描文档中获得的矢量嵌入质量严格依赖于输入图像的质量。该集成要求一个强大的“图像预处理与增强”模块，以便在扫描数据进入OCR引擎前对其混乱性进行规范化。4.1 高保真光栅化策略扫描流程的第一步是将 PDF 页面转换为可作的图像。研究要求使用（包裹 Poppler）以特定配置来平衡画质和内存使用。pdf2imageDPI 选择：标准文本的基础分辨率为 300 DPI，确保字符笔画在 OCR 引擎卷积网络中足够区分。对于包含细则或手写的文档，架构支持扩展至 400-600 DPI。生成器模式：为了防止处理大型手册时出现内存不足（OOM）错误（在“专家”领域中常见），光栅化必须使用 Python 生成器（）。这确保了任何时刻只有一个页面的位图存储在内存中，避免了加载整个 PDF 到内存时出现的巨大内存峰值。yield4.2 几何修正（Deskewing）扫描后的文档不可避免地会出现“偏斜”——由机械扫描器信号引起的旋转错位。即使是偏差$\pm 1^\circ$OCR中的线段算法可能导致文本行被切断，导致乱码。该架构拒绝了计算成本高的霍夫变换，转而采用使用OpenCV的最小面积矩形方法：Binarization & Dilation: The image is converted to a binary map, and a horizontal dilation kernel ($30 \times 5$) is applied. Detailed Implementation ModulesThis section provides the comprehensive code logic for the new modules, ensuring exact adherence to the functionalities described in the Scanned PDF Data Processing & Storage report.6.1 Module 1:  (The Vision Stack)ImageProcessorThis static utility class encapsulates the OpenCV transformations. It is stateless, ensuring thread safety if the ingestion is parallelized in the future.Pythonclass ImageProcessor:
    """
    Implements Image Preprocessing & Enhancement from.
    Handles Deskewing, Rasterization, and Adaptive Binarization.
    """
    @staticmethod
    def deskew(image):
        """
        Applies Minimum Area Rectangle deskewing.
        Source:  Chapter 2.2
        """
        try:
            gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
            gray = cv2.bitwise_not(gray) # Invert: Text becomes white
            
            # Expansion: Dilation to connect text characters
            thresh = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY | cv2.THRESH_OTSU)
            coords = np.column_stack(np.where(thresh > 0))
            
            if coords.size == 0: return image # Handle blank pages
            
            # Calculate angle
            angle = cv2.minAreaRect(coords)[-1]
            if angle < -45: angle = -(90 + angle)
            else: angle = -angle
            
            # Safety Clamp: Only correct if skew is significant but not absurd
            if abs(angle) > 5 or abs(angle) < 0.1: return image
            
            (h, w) = image.shape[:2]
            center = (w // 2, h // 2)
            M = cv2.getRotationMatrix2D(center, angle, 1.0)
            
            # BORDER_REPLICATE is critical to avoid OCR artifacts at edges
            return cv2.warpAffine(image, M, (w, h), 
                                flags=cv2.INTER_CUBIC, 
                                borderMode=cv2.BORDER_REPLICATE)
        except Exception as e:
            print(f"Deskew Warning: {e}")
            return image

    @staticmethod
    def enhance_contrast(image):
        """
        Applies Adaptive Gaussian Thresholding for shadow removal.
        Source:  Chapter 2.3
        """
        try:
            gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
            # Denoise first to prevent grain amplification
            gray = cv2.fastNlMeansDenoising(gray, h=3)
            # Block size 11, C=2 empirically derived for book scans
            binary = cv2.adaptiveThreshold(gray, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, 
                                         cv2.THRESH_BINARY, 11, 2)
            return cv2.cvtColor(binary, cv2.COLOR_GRAY2BGR)
        except Exception:
            return image

    """
    Implements Track A (Structure) & Track B (Semantics).
    Source:  Chapter 3 & 4
    """
    def __init__(self, openai_key=None):
        # Force CPU for Paddle to avoid CUDA OOM with BGE-M3
        self.layout_engine = PPStructure(show_log=False, image_orientation=True, use_gpu=False)
        self.vlm_client = OpenAI(api_key=openai_key) if openai_key else None

    def process_page(self, image_np):
        # 1. Preprocessing
        clean_img = ImageProcessor.deskew(image_np)
        clean_img = ImageProcessor.enhance_contrast(clean_img)
        
        # 2. Layout Analysis (Track A)
        try:
            results = self.layout_engine(clean_img)
        except Exception:
            return

        extracted_data =
        # Sort by Y-coordinate for reading order
        results.sort(key=lambda x: x['bbox'])

        for region in results:
            r_type = region['type']
            bbox = region['bbox']
            
            # Filter Noise 
            if r_type in ['header', 'footer']: continue
            
            content = ""
            # Track A: Text Extraction
            if r_type in ['text', 'title']:
                if 'res' in region:
                    content = " ".join([line['text'] for line in region['res']])
            
            # Track B: Visual Semantics
            elif r_type in ['figure', 'table'] and self.vlm_client:
                content = self._analyze_visual(region['img'], r_type)
            
            if content:
                extracted_data.append({
                    "content": content,
                    "bbox": bbox,
                    "type": r_type
                })
        return extracted_data

    def _analyze_visual(self, img_crop, v_type):
        # Base64 encoding for VLM
        _, buffer = cv2.imencode('.jpg', img_crop)
        b64_img = base64.b64encode(buffer).decode('utf-8')
        
        prompt = "Convert to Markdown." if v_type == 'table' else "Summarize trends."
        
        try:
            response = self.vlm_client.chat.completions.create(
                model="gpt-4o",
                messages=[{
                    "role": "user", 
                    "content": [
                        {"type": "text", "text": prompt},
                        {"type": "image_url", "image_url": {"url": f"data:image/jpeg;base64,{b64_img}"}}
                    ]
                }],
                max_tokens=500
            )
            return f"[{v_type.upper()}]: {response.choices.message.content}"
        except:
            return ""
